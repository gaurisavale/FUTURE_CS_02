# Phishing Email Detection & Awareness System

## Overview
This project was completed as part of the Future Interns Cyber Security Internship Program.

## Objective
To analyze phishing communications, identify phishing indicators, classify risks, and create awareness guidelines for users.

## Tools Used
- Google Phishing Quiz (Jigsaw)
- MXToolbox Header Analyzer
- Gmail
- Microsoft Word
- GitHub

## Methodology
1. Collected phishing and legitimate email examples.
2. Analyzed sender domains and message content.
3. Identified phishing indicators.
4. Classified messages according to risk.
5. Performed email header analysis using MXToolbox.
6. Developed prevention and awareness guidelines.

## Deliverables
- Phishing Detection & Awareness Report
- Email Analysis Examples
- Header Analysis Evidence
- Security Awareness Guidelines
