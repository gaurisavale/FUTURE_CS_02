# Phishing Email Samples

This folder contains sample communications analyzed during the Phishing Email Detection & Awareness System project.

## Sample 1 – Dropbox Storage Notification
Classification: Safe

Indicators:
- Legitimate sender domain (dropboxmail.com)
- Secure HTTPS links
- No credential request

## Sample 2 – Verification Code Request
Classification: Phishing

Indicators:
- Attempts to obtain MFA verification code
- Social engineering tactics
- Account takeover risk

## Sample 3 – Look-Alike Domain Attack
Classification: Phishing

Indicators:
- Fake sender domain
- Impersonation of Google
- Urgency and fear tactics

## Sample 4 – Government Warning Credential Theft
Classification: Phishing

Indicators:
- Fake security warning
- Redirects to credential harvesting page
- Social engineering

## Sample 5 – UPS Delivery Notification
Classification: Safe

Indicators:
- Legitimate delivery service notification
- No credential request
- Valid service links
